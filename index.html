Just a Few Icons

SVG Icons With Some Style

Using the Icons

Fork the repo on Github or install via Bower. Use all the icons in sprite or as inline images. I would recommend you use these with the symbol method for SVG. If you're using Gulp to build your icon sprite here's a reference. More details one Github.

Get the Icons

bower install --save justafewicons
